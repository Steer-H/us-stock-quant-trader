# 交易指令格式规范

> 系统: 美股量化交易系统  
> API: `GET /api/signals`  
> 更新: 2026-06-13

---

## API 响应结构

```json
{
  "signals_md": "Markdown格式文本，每行一条指令",
  "signals": [
    {
      "time": "09:30:01",
      "action": "BUY",
      "ticker": "AAPL",
      "qty": 50,
      "price": 175.00,
      "reason": "开盘建仓"
    }
  ]
}
```

---

## 单条指令格式

```
**{emoji} {action}** {ticker} {qty}股 @ ${price} | {time} | {reason}
```

### 字段说明

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `emoji` | str | 🟢买入 / 🔴卖出 | `🟢` |
| `action` | str | BUY 或 SELL | `BUY` |
| `ticker` | str | 美股代码 | `AAPL` |
| `qty` | int | 股数 | `50` |
| `price` | float | 成交单价(USD, 2位小数) | `175.00` |
| `time` | str | 执行时间 HH:MM:SS | `09:30:01` |
| `reason` | str | 触发条件 | `开盘建仓` |

---

## 六种交易原因

| 原因 | 触发条件 | 概率 |
|------|----------|------|
| `开盘建仓` | 美东 09:30 首次持仓配置 | 100% |
| `短线止盈({x}%)` | 浮动盈亏 ≥ +3% | 80% |
| `短线止损({x}%)` | 浮动盈亏 ≤ -4% | 90% |
| `ML预测卖出` | 盈亏 < 1% 且模型信号偏空 | 17.5% |
| `超时平仓({x}分钟)` | 单笔持仓 > 30分钟 | 100% |
| `ML短线买入` | 动量信号 + 现金 > $5000 | 按信号 |

## 仪表盘渲染

指令卡片样式：
- 字体: 等宽 (`SF Mono / Monaco / Menlo`)
- 字号: 12px
- 🟢 BUY 行: 绿色加粗
- 🔴 SELL 行: 红色加粗
- 股票代码: 白色加粗
- 价格/时间/原因: 灰色辅助文字
- 滚动: 最大高度 300px，超出可滚动
- 刷新: 每 3 秒自动拉取
